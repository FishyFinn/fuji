\LevelOne{command\_interactive}
\LevelTwo{Purpose}
This module allows you to write commands in \tbf{sign block}.

\LevelTwo{How it works?}
When a player \ttt{right click} a \ttt{sign block}, this module will check if the sign block contains the character \str{/}.
If contains, then we \ttt{treat as} the player issue the command followed by the character.

\LevelTwo{Definition}
A \ttt{sign block} that contains the character \str{/} is named \ttt{an interactive sign block}.

\begin{note}{How can i edit an interactive sign block?}
    You need to press \ttt{shift + right click} to edit an interactive sign.
\end{note}

\LevelTwo{Example}
\begin{example}{Basic usage}
    /say hi \ph{player:name}\\
    line 2 empty\\
    line 3 empty\\
    line 4 empty
\end{example}

\begin{example}{Add a prefix description text}
    prefix /say first\\
    /say the second\\
    /say hi \ph{player:name}\\
    /say the last command
\end{example}

\begin{example}{Concat commands between lines}
    prefix /say this is\\
    the first /say and the\\
    second\\
    line 4 empty
\end{example}
