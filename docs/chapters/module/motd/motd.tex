\LevelOne{motd}
\LevelTwo{Purpose}
This module provides motd customization.

\LevelTwo{Example}
\begin{example}{Configure server icons}
    You can put 64x64 pixels \ttt{.PNG images} into the directory \ttt{config/fuji/modules/motd/icon}, this module will pick up a random image as the icon of the server to respond the server status request.
\end{example}
